2. Description of Service

The Link Builder is a browser-based utility that enables users to generate deep-link URLs for Bynder Digital Asset Management portals. The tool supports URL construction based on metaproperties, option values, tags, keywords, and asset status filters. An integrated asset browser is available to authenticated Bynder users, enabling direct retrieval of asset metadata to assist with link construction.

All URL generation is performed entirely client-side. No data entered by the user is transmitted to, processed by, or stored on any server operated by or on behalf of this service.

3. Data Processing & Privacy

This website is designed with a minimal data footprint. The following applies to all users:

  • Third-party services — This website does not engage any third-party services for advertising, user profiling, or data monetisation purposes.
